## 🏡 House Price Prediction in Ho Chi Minh City (2023)

Final project for the "Introduction to Data Science" course at HCMUS.

📌 **Goal**: Predict house/apartment prices in Ho Chi Minh City using regression models.  
📊 **Data**: Crawled from [batdongsan.vn](https://batdongsan.vn) via web scraping.  
🧪 **Models**: Linear, Ridge, Lasso, Random Forest, Gradient Boosting  
📈 **Best Models**: Gradient Boosting & Random Forest (based on RMSE, MAE, R²)  
🛠️ **Tech**: Python, Scikit-learn, Pandas, PCA, BeautifulSoup
